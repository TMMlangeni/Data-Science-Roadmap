# Investigating Netflix Movies: 1990s Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix movie data to uncover trends from the 1990s. As a final-year student at NWU, I utilized this project to demonstrate proficiency in Python-based data manipulation, moving beyond academic theory into practical, recruiter-ready applications.

## Key Objectives
* **Data Inspection**: Processed the `netflix_data.csv` dataset using Pandas.
* **Modal Analysis**: Identified the most frequent movie duration in the 1990s.
* **Advanced Filtering**: Applied vectorized boolean masks to isolate "Action" movies under 90 minutes.
* **Portfolio Development**: Structured the analysis for inclusion in a professional Data Science portfolio.

## Technical Stack
* **Language**: Python
* **Libraries**: Pandas, Matplotlib
* **Methodologies**: Vectorization, Boolean Indexing, Data Cleaning.

## Project Findings
* **1990s Movie Duration**: Successfully calculated the most frequent movie length for the decade.
* **Action Movie Trends**: Determined the exact count of short-form Action movies (under 90 minutes) released between 1990 and 1999.

## How to Run
1. Ensure you have Python and Pandas installed.
2. Download the `netflix_data.csv` and the Jupyter Notebook.
3. Run the notebook to reproduce the analysis and visualizations.
